# 打包体积报告:那 248MB 怎么办

## TL;DR

**安装包不带那个二进制,首次启动时后台下载。** Windows 安装包从 ~160MB 降到 ~85MB,
用户点开图标 2 秒就能看到界面,248MB 在后台悄悄下。

## 实测数字

| 项 | 大小 | 来源 |
|---|---|---|
| `@anthropic-ai/claude-agent-sdk`(核心 JS) | **3.8 MB** | `du -sh` 实测 |
| `claude-agent-sdk-linux-x64/claude`(原生二进制) | **248 MB** | `du -sh` 实测 |
| ↑ 同一个二进制 gzip -9 之后 | **76.3 MB** | `gzip -9` 实测 |
| Electron 33 运行时(win32-x64) | **109.7 MB** | GitHub Release 实测 |
| Electron 33 运行时(darwin-arm64) | **95.1 MB** | GitHub Release 实测 |

**先纠正一个直觉误区**:很多人一看到 248MB 就以为安装包会涨 248MB。不会 —— 安装包是
压缩的,这个二进制压完只有 **76MB**。所以"带上它"的真实代价是 **+76MB**,不是 +248MB。

但磁盘占用是躲不掉的:二进制必须解压成可执行文件才能跑,**装完之后无论如何都是 ~350MB**。
我们能优化的只有**安装包大小**和**首次可用时间**。

## 关键发现:二进制是按平台分发的 optionalDependency

```json
"optionalDependencies": {
  "@anthropic-ai/claude-agent-sdk-darwin-arm64": "0.3.209",
  "@anthropic-ai/claude-agent-sdk-win32-x64": "0.3.209",
  ... 共 8 个平台
}
```

意味着:
- **单平台构建只会装 1 份二进制,不是 8 份。** 不用担心 8 × 248MB。
- 但**绝不能出 universal 包** —— 那会把两个架构的二进制都塞进去,直接翻倍。
  `electron-builder.yml` 里已经按 `arch: [arm64, x64]` 分开出包。

## 三个方案的取舍

| 方案 | 安装包 | 首次可用 | 装完磁盘 |
|---|---|---|---|
| A. 二进制打进安装包 | **~160 MB** | 装完即用 | ~350 MB |
| **B. 首次启动后台下载(选它)** | **~85 MB** | **界面秒开**,agent 功能等下载完 | ~350 MB |
| C. 全部上云(不内嵌 SDK) | ~85 MB | 秒开 | ~100 MB |

**C 被产品需求否决了** —— agent 必须能操作用户本地的浏览器/抖音登录态,那就必须在本地跑。

**B 优于 A 的三个理由:**
1. **下载页转化率**:85MB 和 160MB 在下载按钮上的心理差异很大。
2. **agent 运行时可独立升级**:SDK 从 0.3.209 升级到 0.4.x,不需要用户重装整个 app ——
   换个 sha256 和 URL 就行。SDK 迭代很快,这条价值不小。
3. **首屏时间**:用户点开就看到界面,而不是盯着安装进度条。

**B 的代价**(必须处理好,否则会变成事故):
- 首次启动必须联网。**弱网/断网时不能白屏** —— 当前实现:下载失败只打日志,界面照常起,
  agent 功能不可用(`src/main/index.ts` 里 `.catch()` 掉了)。**待办**:UI 上要给出明确
  的"AI 功能准备中 / 下载失败重试"状态,不能让用户对着一个点了没反应的输入框发懵。
- 下载必须**校验 sha256**。见下。

## ⚠️ 安全:这个二进制会以用户权限执行任意代码

按需下载意味着我们在用户机器上**下载并执行**一个可执行文件。下载链路被劫持 = RCE。

所以 `src/main/binary.ts` 里:
- 下载先落 `.part` 临时文件
- **校验 sha256 通过之后**才 rename 到最终路径
- 校验不过就抛错、拒绝执行,宁可 agent 功能不可用

这几条有测试守着(`apps/desktop/test/binary.test.ts`,6 个用例,含"被篡改的下载内容
不会落到最终路径")。**改这块代码前先看那个测试。**

## electron-builder 侧的配置

```yaml
files:
  # 把二进制排除在安装包外 —— 这一行就是 160MB → 85MB
  - '!node_modules/@anthropic-ai/claude-agent-sdk-*/**'
```

如果哪天决定改回方案 A(比如发现首次下载失败率太高),**把这一行删掉即可**,
`binary.ts` 会自动走"本地已存在就直接用"的分支,不用改代码。
