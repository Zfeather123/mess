# 小镜桌面版 · 团队内部安装指南

> 开发阶段我们**没有买代码签名证书**(产品决策)。所以包是**未签名**的,系统一定会拦你一下。
> **这是正常的,不是软件坏了。** 按下面点几下就能装上。

## 第一步:下载安装包

1. 打开 GitHub 上的 [Actions → Desktop (Win/macOS)](https://github.com/Zfeather123/mess/actions/workflows/jin-desktop.yml)
2. 点最上面那条**绿色✅**的记录
3. 拉到页面最下面,**Artifacts** 区域:
   - Windows 下载 `xiaojing-desktop-windows`
   - Mac 下载 `xiaojing-desktop-macos`
4. 下载下来是个 zip,解压出来才是安装包

> 需要 GitHub 账号且有这个仓库的权限。没有的话找 Yu Zheng 加一下。

---

## Windows 安装

双击安装包,会弹一个**蓝色**的窗:

> **Windows 已保护你的电脑**
> Microsoft Defender SmartScreen 阻止了无法识别的应用启动…

它默认只给你一个「不运行」按钮。**别慌,真正的按钮藏起来了:**

1. 点左边那行小字 **「更多信息」**
2. 这时下面会多出一个按钮 **「仍要运行」** ← **点它**
3. 正常安装

---

## macOS 安装

把 `小镜.app` 拖进「应用程序」后,**第一次双击会被 Gatekeeper 拦住**,弹窗往往只给「移到废纸篓」——**不要点废纸篓。**

### 办法一(推荐,不用碰终端)

1. 关掉那个弹窗
2. 打开 **系统设置 → 隐私与安全性**
3. **往下滚**,在「安全性」那一段会看到一行:
   > 「已阻止使用"小镜",因为它来自身份不明的开发者」
   旁边有个按钮 **「仍要打开」** ← **点它**
4. 再弹一次确认框,点 **「打开」**。以后就能正常双击了。

> ⚠️ **注意**:老教程里说的「右键 →打开」在 **macOS Sequoia(15)及以后已经被苹果移除了**,不要再找那个菜单,找不到的。**只能走「系统设置」这条路。**

### 办法二(终端一行搞定)

```bash
xattr -cr /Applications/小镜.app
```

这行是去掉「从网上下载」的隔离标记。跑完直接双击就能开。

## 常见问题

**Q:提示「小镜已损坏,无法打开」怎么办?**
A:多半也是隔离标记的问题,不是真的损坏。跑上面的 `xattr -cr` 那行。

**Q:为什么装个自家软件这么麻烦?**
A:因为没签名。签名要买证书(Apple 99 美元/年 + Windows 证书),**开发阶段先不花这个钱**。等要给第一个真实用户之前会买,到时候这些步骤全都不需要了。

**Q:这样绕过安全提示,安全吗?**
A:对**这个包**是安全的——它是从我们自己的 GitHub 仓库、由 CI 从源码构建出来的,构建记录可查。但请**只装从上面那个 Actions 页面下载的包**,别人私发给你的 zip 不要装。
